NavHost

@Composable
fun <C : Any> NavHost(startingNavControllerInstance: NavController<C>, modifier: Modifier = Modifier, excludedDestinations: List<C>? = null, animations: DestinationAnimationsConfiguratorScope<C>.() -> ContentAnimations = LocalContentAnimator.current, router: @Composable AnimatedVisibilityScope.(destination: C) -> Unit)

Sets up a stack animator and manages back gestures for the animations. Provides each destination with ComponentContext via LocalComponentContext. Animations are passed down using CompositionLocalProvider for other NavHosts to consume.

Example:

@Composable
fun MyScreen(navController: NavController<String>) {
    NavHost(
	       startingNavControllerInstance = navController,
		   animations = { cleanSlideAndFade() }
	    ) { destination ->
		    when (destination) {
			    // handle destinations here
		   }
	    }
}

Parameters

startingNavControllerInstance

The NavController instance for the navigation host.

modifier

An optional modifier for the host container.

excludedDestinations

A list of destinations that should not be rendered or animated.

animations

A lambda configuring the animations for the stack. Defaults to cleanSlideAndFade().

router

A composable function that receives the current destination and renders the corresponding content.
